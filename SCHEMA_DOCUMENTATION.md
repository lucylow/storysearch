```typescript

}
```

## üîç Field Specifications

### **id** (required)
```typescript
type: string
format: any
example: "nike-product-1"
purpose: Unique identification for search indexing
```

**Best Practices:**
- Use kebab-case format
- Include brand prefix
- Include content type
- Make descriptive

**Examples:**
- ‚úÖ `nike-product-air-max-90`
- ‚úÖ `blackpink-music-born-pink`
- ‚úÖ `ishowspeed-video-fifa-25`
- ‚ùå `1` (too generic)
- ‚ùå `product` (not unique)

### **title** (required)
```typescript
type: string
format: any
example: "Nike Air Zoom Pegasus 40"
purpose: Main heading for display and search
```

**Best Practices:**
- Clear and descriptive
- Include brand name when relevant
- Use proper capitalization
- 40-80 characters ideal

### **slug** (required)
```typescript
type: string
format: kebab-case
example: "nike-air-zoom-pegasus-40"
purpose: URL-friendly identifier
```

**Best Practices:**
- lowercase only
- hyphens for spaces
- remove special characters
- match title semantically

### **content_type** (required)

type: string
format: any
examples:
